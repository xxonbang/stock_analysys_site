# 일동제약 종목 분석 제외 문제 분석

## 문제 상황

'일동제약' 종목을 분석 요청했는데 결과 리포트에서 제외되었습니다.

## 원인 분석

### 1. 티커 변환 실패
- `KOREA_STOCK_MAP`에 '일동제약'이 없음
- `normalizeStockSymbol`이 변환 실패 시 원본('일동제약')을 그대로 반환

### 2. 데이터 수집 실패
- Python 스크립트에서 '일동제약'을 한국 주식으로 인식
- FinanceDataReader는 티커가 필요한데, 한글 이름으로는 직접 조회 불가
- Python 스크립트의 이름→티커 변환 로직이 `StockListing` API 오류로 실패

### 3. 결과 제외
- 데이터 수집 실패 → `stockDataMap.get('일동제약')`이 `undefined`
- `if (!stockData) { continue; }`로 인해 해당 종목이 결과에서 제외

## 해결 방법

### 즉시 해결
1. ✅ '일동제약' 티커를 `KOREA_STOCK_MAP`에 추가 (티커: 249420)

### 근본적 해결
1. Python 스크립트의 `StockListing` 오류 처리 개선
2. TypeScript 레벨에서도 이름→티커 변환 시도 (API 호출)
3. 데이터 수집 실패 시에도 에러 메시지를 포함한 결과 반환
